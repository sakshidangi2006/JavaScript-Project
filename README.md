# 🚀 JavaScript Projects Collection

A curated collection of beginner-to-intermediate JavaScript projects built with vanilla HTML, CSS, and JavaScript. Each project demonstrates core web development concepts and is fully self-contained.

---

## 📁 Projects Overview

| Project | Description | Key Concepts |
|---|---|---|
| [Calculator](#-calculator-program) | A functional arithmetic calculator | DOM manipulation, event handling |
| [Counter](#-counter-program) | Increment/decrement counter app | State management, UI updates |
| [Kanban Board](#-kanban-board) | Drag-and-drop task manager | Local storage, drag & drop API |
| [Random Jokes](#-random-jokes) | Fetches and displays random jokes | Fetch API, async/await |
| [Snake Water Gun Game](#-snake-water-gun-game) | Rock-Paper-Scissors variant | Game logic, conditionals |
| [Temperature Conversion](#-temperature-conversion) | Converts between temperature units | Form handling, math operations |
| [User Directory](#-user-directory) | Searchable user listing | API integration, filtering |

## 🧮 Calculator Program

A clean, fully functional calculator that supports basic arithmetic operations.

**Features:**
- Addition, subtraction, multiplication, and division
- Clear and delete functionality
- Decimal number support
- Keyboard-friendly UI

**Tech Stack:** HTML, CSS, JavaScript

---

## 🔢 Counter Program

A simple yet elegant counter application with interactive controls.

**Features:**
- Increment and decrement buttons
- Reset functionality
- Visual feedback for positive/negative values

**Tech Stack:** HTML, CSS, JavaScript

---

## 📋 Kanban Board

A drag-and-drop task management board inspired by tools like Trello.

**Features:**
- Create, edit, and delete tasks
- Drag tasks across columns (To Do → In Progress → Done)
- Persistent data via `localStorage`
- Color-coded task cards

**Tech Stack:** HTML, CSS, JavaScript (Drag & Drop API, localStorage)

---

## 😂 Random Jokes

A fun app that fetches and displays random jokes from a public API.

**Features:**
- Fetch a new joke on button click
- Displays setup and punchline format
- Loading state while fetching
- Responsive card layout

**Tech Stack:** HTML, CSS, JavaScript (Fetch API, async/await)

---

## 🐍 Snake Water Gun Game

A fun twist on the classic Rock-Paper-Scissors game using Snake, Water, and Gun.

**Rules:**
- 🐍 Snake drinks Water → Snake wins
- 💧 Water douses Gun → Water wins
- 🔫 Gun kills Snake → Gun wins

**Features:**
- Player vs. Computer gameplay
- Randomized computer choice
- Score tracking
- Win/lose/draw result display

**Tech Stack:** HTML, CSS, JavaScript

---

## 🌡️ Temperature Conversion

A utility app that converts temperatures between different units instantly.

**Features:**
- Converts between Celsius, Fahrenheit, and Kelvin
- Real-time conversion on input
- Clean, minimal interface

**Tech Stack:** HTML, CSS, JavaScript

---

## 📒 User Directory

A searchable directory that fetches and displays user data from an API.

**Features:**
- Fetches user data from a REST API
- Live search/filter by name
- Displays user cards with details (name, email, phone, etc.)
- Responsive grid layout

**Tech Stack:** HTML, CSS, JavaScript (Fetch API)

## 🛠️ Getting Started

Each project is standalone. To run any project locally:

```bash
# 1. Clone the repository
git clone https://github.com/sakshidangi2006/<repo-name>.git

# 2. Navigate into the desired project folder
cd Calculator-Program

# 3. Open index.html in your browser
open index.html
```

> No build tools or dependencies required — pure HTML, CSS, and JavaScript!
